UPNMThesis, a LaTeX class for typesetting a Universiti Pertahanan Nasional
Malaysia (UPNM) thesis.
    Copyright (C) 2015  Lim Lian Tze

    PLEASE READ upnmthesis-manual.pdf.
    You are recommended to use sample-thesis.tex as a starting template.

    Files:
    upnmthesis.cls           LaTeX class file
    upnmthesis-manual.pdf    A quick user guide
    sample-thesis.tex        Example 'main' thesis .tex file
    *.tex                    Other component .tex files (chapters etc)
    sample-thesis.pdf        Example PDF file of a UPNM thesis, can be
                             generated from thesis.tex and friends
    myrefs.bib               Sample BibTeX bibliography database file
